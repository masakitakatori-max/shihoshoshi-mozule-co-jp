LEC（東京リーガルマインド）は司法書士講座の老舗で、40年以上にわたり多数の合格者を輩出してきた最大手予備校です。しかし受講料が高額なため、本当にそれだけの価値があるのか気になる方も多いでしょう。この記事ではLEC司法書士講座の良い評判・悪い評判を受講者の口コミから徹底検証します。

LEC司法書士講座の基本情報

| 項目 | 内容 | |------|------| | 運営会社 | 株式会社東京リーガルマインド | | 講座実績 | 40年以上（司法書士講座のパイオニア） | | 合格実績 | 累計合格者数業界トップクラス | | 受講料 | 約35〜50万円（コースによる） | | 学習スタイル | 通信＋通学（全国30校以上） | | 看板講師 | 海野禎子講師、森山和正講師ほか | | 質問対応 | あり（回数制限付き） | | 受講期間 | 15〜21ヶ月 |

LECの良い評判・口コミ

評判1：講師の質が圧倒的に高い 「海野講師の講義は本当にわかりやすい。複雑な登記法の制度趣旨から丁寧に説明してくれるので、暗記ではなく理解で解ける力がついた。この講師に出会えたことが合格の最大の要因」（30代男性・合格者）

評判2：記述式対策が業界最強 「LECの記述式対策は他社と比べて圧倒的に充実している。添削指導が丁寧で、自分では気づかない書き方の癖や減点ポイントを具体的に指摘してくれた」（40代女性・合格者）

評判3：通学・通信を自由に切り替えられる 「基本は通信で受講しているが、記述式の答練は通学で受けるなど、柔軟に使い分けられるのが便利。自習室も全国の校舎で使えるので、集中したいときに利用していた」（20代男性・社会人）

評判4：圧倒的な合格実績の安心感 「司法書士講座といえばLECという安心感がある。合格者の多くがLEC出身であり、合格者同士のネットワークも強い」（30代女性・合格者）

LECの悪い評判・口コミ

評判5：受講料が高い 「新15ヵ月合格コースは約45万円と高額。スタディングの約5万円と比べると9倍の差がある。内容は確かに充実しているが、学生には厳しい金額」（20代男性・学生）

評判6：カリキュラムの量が多すぎる 「講義時間が500時間以上あり、全部消化するのが大変。仕事をしながらだとどうしても講義の消化が遅れがちになる」（30代男性・社会人）

評判7：テキストのデザインが古い 「テキストが白黒中心で見た目が地味。アガルートのフルカラーテキストと比べると視覚的な魅力に欠ける。内容は素晴らしいのだが」（20代女性・初学者）

LECが向いている人・向いていない人

向いている人：

  • 記述式対策を万全にしたい人
  • 通学で受講したい人（全国30校以上）
  • 老舗予備校の実績と安心感を求める人
  • 自習室を使いたい人

向いていない人：

  • 費用を抑えたい人→スタディング
  • コンパクトなカリキュラムを求める人→クレアール
  • スマホ学習中心にしたい人→スタディング

まとめ

LEC司法書士講座は「記述式対策の充実度」と「40年以上の実績」が最大の強みです。受講料は高額ですが、確実に合格を掴み取りたい人にとっては最も信頼できる選択肢です。まずは無料の公開講座やガイダンスに参加して、講師の講義スタイルを体験してみましょう。
