2026年（令和8年）の司法書士試験は7月第1日曜日に筆記試験、10月中旬に口述試験が実施される見込みです。申込期間は5月上旬〜中旬の約2週間と短いため、早めの準備が必要です。この記事では2026年の試験スケジュール、申込方法、当日の注意点を解説します。

2026年 司法書士試験の日程（予定）

| イベント | 日程（予定） | 備考 | |---------|------------|------| | 受験案内配布 | 2026年4月上旬 | 法務局窓口 or 法務省HPでダウンロード | | 受験申込期間 | 2026年5月上旬〜中旬 | 約2週間。郵送は消印有効 | | 筆記試験 | 2026年7月5日（日） | 午前の部＋午後の部 | | 筆記試験合格発表 | 2026年9月下旬 | 法務省HP＋法務局掲示 | | 口述試験 | 2026年10月中旬 | 筆記合格者のみ | | 最終合格発表 | 2026年11月上旬 | 法務省HP＋官報 |

※正式な日程は法務省から発表される試験公告（2026年4月頃）で確認してください。

試験当日のスケジュール

| 時間 | 内容 | |------|------| | 9:00〜9:30 | 入室・着席 | | 9:30〜11:30 | 午前の部（択一式35問・2時間） | | 11:30〜13:00 | 昼休み | | 13:00〜16:00 | 午後の部（択一式35問＋記述式2問・3時間） |

午前の部（9:30〜11:30） 民法20問、商法（会社法）9問、憲法3問、刑法3問の計35問を2時間で解きます。1問あたり約3.4分のペース配分が必要です。

午後の部（13:00〜16:00） 択一式35問と記述式2問を3時間で解きます。時間配分が最大の課題で、択一に60〜70分、記述式に110〜120分が目安です。司法書士の試験科目で各科目の出題数を確認しておきましょう。

申込方法と必要書類

必要書類：

  1. 受験申請書（法務局で入手 or ダウンロード）
  2. 受験手数料：8,000円分の収入印紙
  3. 写真（縦5cm×横5cm、6ヶ月以内撮影、裏面に氏名記入）
  4. 本人確認書類の写し（運転免許証、マイナンバーカード等）
  5. 返信用封筒（受験票送付用）

申込方法：

  • 窓口提出： 受験地を管轄する法務局に直接持参
  • 郵送： 書留郵便で送付（消印有効）

注意点：

  • 申込期間は約2週間と非常に短い
  • 期間を過ぎると受験不可（追加申込なし）
  • 受験地は原則変更不可

試験会場

司法書士試験は全国の法務局所在地で実施されます。主な試験会場は以下のとおりです。

| 管轄法務局 | 試験地 | |-----------|--------| | 東京法務局 | 東京都内 | | 大阪法務局 | 大阪市内 | | 名古屋法務局 | 名古屋市内 | | 福岡法務局 | 福岡市内 | | 仙台法務局 | 仙台市内 | | 札幌法務局 | 札幌市内 | | 広島法務局 | 広島市内 | | 高松法務局 | 高松市内 |

具体的な会場は受験票に記載されます。

合格発表と合格後の流れ

筆記試験合格発表（9月下旬）： 法務省HP、法務局掲示で発表されます。合格者には口述試験の案内が届きます。

口述試験（10月中旬）： 筆記試験合格者のみ受験できます。面接形式で、不動産登記法・商業登記法・司法書士法に関する口頭試問が行われます。口述試験の合格率は約99%であり、筆記に合格すればほぼ最終合格できます。

最終合格発表（11月上旬）： 合格後は日本司法書士会連合会への登録が必要です。

試験までの学習スケジュール

2026年7月の試験に向けた残り期間別の学習指針です。

| 残り期間 | 状況 | 対策 | |---------|------|------| | 12ヶ月以上 | 初学者が始めるのに最適 | 勉強法に沿って計画的に | | 6〜12ヶ月 | 学習経験者なら間に合う | 弱点科目の強化と記述式対策を集中的に | | 3〜6ヶ月 | 短期合格は難しい | 翌年の受験を視野に基礎固めを優先 | | 3ヶ月未満 | 今年は見送り推奨 | 翌年に向けた長期計画を立てる |

司法書士の難易度を考慮すると、初学者は最低18ヶ月前からの学習開始が理想です。
